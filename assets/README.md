# mockup/assets — 背景動画・画像の置き場

TOP（top.html）のヒーロー背景動画は、このフォルダのファイルを参照します。

## 置くファイル
- `hero.mp4` … 背景動画（御社制作のもの）
  - 推奨：横長（16:9）／10〜20秒ループ／**5MB前後**（重いと表示が遅くなる）
  - 音声なし（自動再生は muted 前提）
  - 可能なら `hero.webm` も併用すると軽く・互換性UP
- `hero-poster.jpg` … 動画読み込み前/非対応時に表示する静止画（任意）

## 動作
- `hero.mp4` を置くと、TOPのヒーローで**自動再生・ループ**します。
- ファイルが無い場合は、CSSの「ゆっくりズームする背景（Ken Burns）」が代わりに表示されます（壊れません）。

## 差し替え手順
1. このフォルダに `hero.mp4`（と任意で `hero-poster.jpg`）を置く
2. top.html をブラウザで再読み込み

> 本番（WordPress/SWELL）でも同じ考え方で、ヒーローブロックに動画背景を設定します。
